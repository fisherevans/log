I made this tool to aid in creating perfectly looped and stabilized GIFs. From time to time I'll take a video with my phone hoping that someday I'd turn it into a GIF and make it a perfect loop. I made this little app to avoid the manual task of shifting frame by frame so things lined up, and solved it by letting the user define "animation anchors" - the app interpolates from anchor to anchor to create smoother loops.

I originally made it for personal use so it's not polished and there are some quirks. Let me know if you like the idea and maybe I'll clean it up some more or add more features.

[Source on GitHub](https://github.com/fisherevans/GIFLooper)

Originally posted to [/r/perfectloops](https://www.reddit.com/r/perfectloops). I had a handful of sample before/after GIFs (Gears #1 and #2) hosted on Gfycat, which has since shut down - so those links are dead.
